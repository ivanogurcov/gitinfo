Клонирование проекта в папку nameProject:
git clone https://...nameProject

Добавить файлы коммит:
git add .

Записать коммит:
git commit -m "Комментарий коммита"

Отправить коммит:
git push

Принять изменения:
git pull
Авторство коммитов:
git config --global user.email "Ваш емайл"
git config --global
